小米6刷机Mokee+Google服务框架体验Android原生

发表于 2019-06-17 | 更新于 2019-06-20
本文字数： 818 | 阅读时长 ≈ 1 分钟

背景

最近学英语背单词，发现ios设备因为权限问题有很多限制。比如“百词斩”在Android平台有替换锁屏界面为单词提示的功能，ios平台则无法实现。
正好手头有一台小米6设备闲置，无奈MIUI广告太多。很早前就想体验Mokee开源系统接近原生Android的体验。决定刷机Mokee+Google框架，尝鲜原生Android。

步骤

  • 解锁手机

    1. 申请 - 小米账号解锁权限
    2. 获得解锁资格后 - 下载小米解锁工具
    3. 手机操作解锁
      1. 进入“设置 -> 开发者选项 -> 设备解锁状态”中绑定账号和设备；
      2. 手动进入Bootloader模式（关机后，同时按住开机键和音量下键）；
      3. 通过USB连接手机，点击 “解锁”按钮；

  • 刷入recovery

    1. 进入魔趣下载页搜索设备名
    2. 点击Recovery下载 下载Recovery刷机包
    3. 使用小米刷机工具包中的工具 命令行执行命令验证设备

      fastboot devices

    4. 刷入Recovery

      fastboot flash recovery twrp-x.x.x-x-x.img

    5. 进入Recovery

      关闭设备后，按住音量调高 + 电源键，直到进入 Recovery 模式，然后松开
      // 注意此处一定按住按键，出现twrp引导页再松开，否则会进入小米自带Recovery

  • 刷入ROM OpenGapps Magisk

    1. 下载ROM刷机包 此处可提前下载放入手机sdcard目录
    2. Recovery下点击WIPE
    3. 点击 Format Data 执行格式化过程。这将删除加密以及存储在内部存储上的所有文件
    4. 找到刷机包，刷入系统
    5. 可同样刷入OpenGapps（Google框架）以及Magisk Manager（Root权限）

      注意：此处手机不可设置锁屏密码，否则在Recovery下要求输入密码，导致无法解锁Data，找不到下载的刷机包。
